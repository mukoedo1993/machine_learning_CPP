Basic linear classifier.

The LinearClassifier class is a multi class classifier model suited for linear discriminant analysis. For c classes 0,…,c−1 the model computes

argmaxiwTix+bi
Thus is it a linear model with arg max computation. The internal linear model can be queried using decisionFunction().
