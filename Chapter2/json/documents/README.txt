It is a kind of data representation format very similar to XML.
It is commonly used for APIs and configs.
It is lightweight and easy to read and write.
It integrates very nicely with Javascript since JSON is a subset of Javascript,
so anything you write in JSON is valid in Javascript.
Integrate easily with other languages (including CPP here)
